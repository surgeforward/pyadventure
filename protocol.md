# PyPeg2 Setup

## 1. Set up virtualenv

Using Terminal:

```
> cd project_root
virtualenv env
source env/bin/activate
```

## 2. Install PyPEG2

Note: PyPeg2 is a *Python 3* library.

```
pip install pypeg2
```

## 3. Import the library 

```python
from pypeg2 import *
```

## 4. Voila!

For further documentation, check out [the official site.](https://fdik.org/pyPEG/)
